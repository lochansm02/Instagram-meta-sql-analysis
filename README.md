## 📌 Project Overview

 📸 Instagram (Meta) SQL Data Analysis Project

This project analyzes an Instagram (Meta)-style database using **MySQL** to solve real-world business problems through SQL. The analysis focuses on user engagement, content performance, hashtag usage, and platform activity by writing optimized SQL queries and interpreting the results with charts and business insights.

The project demonstrates practical SQL skills commonly used by Data Analysts and Business Analysts.

---

## 🎯 Business Objective

The objective of this project is to extract meaningful insights from Instagram data that can help the business:

- Understand user activity and engagement.
- Identify top-performing content.
- Analyze hashtag popularity.
- Discover inactive and highly active users.
- Support data-driven business decisions.

## 🗂️ Database Schema

The project uses an Instagram-like relational database containing tables such as:

These tables represent user interactions and relationships within the platform.

## 📊 Project Objectives

The analysis includes solving business questions such as:

- Identifying the oldest users on the platform.
- Finding inactive users who have never posted.

- Analyzing the most popular hashtags.
